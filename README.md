# Kalima
This is an implementation of the popular game Wordle in Python and a Qt interface.

**Requirements**
- Python 3.7+
- PyQt
- Best viewed with a screen resolution of at least `1920x1080`

**Installation**

`pip3 install -r requirements.txt`
